# 智能图片压缩

系统提供图片智能压缩功能，可在系统配置中开启。

![智能图片压缩配置](../../images/smart_image_compression_config.png)

## 压缩规则

- 小于 1MB 的 jpg/png 图片跳过压缩
- webp 图片不压缩（因为已足够小）

### JPEG 图片

| 文件大小 | 压缩质量 |
|---------|---------|
| ≤ 2MB   | 0.55    |
| ≤ 5MB   | 0.45    |
| > 5MB   | 0.35    |

### 其他格式（转 JPEG）

| 文件大小 | 压缩质量 |
|---------|---------|
| ≤ 500KB | 0.7     |
| ≤ 2MB   | 0.6     |
| ≤ 5MB   | 0.5     |
| > 5MB   | 0.4     |

## 处理策略

图片会在第一次加载预览时进行压缩。正常情况下，在编辑器上传后就会进入压缩队列（老版本升级后会在图片第一次被访问时加入压缩队列）。

### 获取原图

如需获取原图，在图片 URL 后面加入 `?download=true`，例如：

```
https://apilinks.cn/file/5561656e3db242a6bf342714f08855ab?download=true
```

文章阅读器的图片查看功能中也添加了**下载原图**功能：
